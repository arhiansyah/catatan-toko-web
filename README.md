
# Catatan Web Project

Personal Project

## Installation

Clone my project

```bash
  clone this project
  cd catatan-toko-web
```

Install with composer
```bash
  composer install
```

Building Asset with npm
```bash
  npm install && npm run dev
```

Setting Env in project, and set your db then running command :
```bash
  php artisan migrate --seed
```
